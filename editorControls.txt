Hold Spacebar + Hold Left Mouse Button: Move Camera
Ctrl + S: Save Level
Tab: Enable/Disable Physics
Select Objects and Ctrl + C: Copy Selected Objects
Select Two Objects and Shift + J: Joint Editor
Select Bird and Press 1 .. 9: Set Bird Launch Order from 1 to 9
L: Indicate Bird Launch Order
Shift + B: Set Bird Camera location and zoom
Shift + C: Set Castle Camera location and zoom
Ctrl + Shift + B: Check Bird Camera location and zoom
Ctrl + Shift + C: Check Castle Camera location and zoom
Ctrl + P: Particle Editor
Shift + P: Play Level!
Select Objects and Enter key + arrow buttons: Allign Selected Objects
O: Indicate Origin
8: Show Collision boxes
0: Draw One Layer
Select Objects and Shift + R/Shift + Ctrl + R: Rotate Objects
Select Objects and Ctrl + Shift + Hold Right Mouse Button: Rotate Objects
Select Objects and Press Arrow keys: Move Objects
Select Objects and Hold Left Mouse Button: Move Objects
Right Mouse Click: Select Objects
Hold Right Mouse Button and Drag: Select Multiple Objects
Shift + Right Mouse Click: Select Multiple Objects
Select Objects and Ctrl + Arrow Key: Mirror Objects


Shift + T: Change Theme
